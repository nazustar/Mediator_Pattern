# Importante: 
Para que el programa ejecute usar:

## 1. Ir al proyecto
cd SmartHome

## 2. Compilar y ejecutar

`mvn clean compile`

`mvn exec:java -Dexec.mainClass="app.Main"`

O desde visual usar el botón run en el main.
